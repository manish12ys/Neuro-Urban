## 🎯 Project Overview

NeuroUrban revolutionizes urban planning by leveraging artificial intelligence to create sustainable, livable, and efficient cities. Our system combines the best practices from world-class cities while preserving cultural heritage and promoting environmental sustainability.

### 🔬 **Core Capabilities**

| Component | Technology | Purpose |
|-----------|------------|---------|
| 🔍 **City Analysis** | Machine Learning + GPU Acceleration | Analyze patterns from 20+ global cities |
| 🏗️ **Blueprint Generation** | GANs + Deep Learning | Generate optimal city layouts |
| 🎮 **City Simulation** | Reinforcement Learning | Simulate city dynamics and policies |
| 📊 **Interactive Dashboard** | Streamlit + Real-time Viz | Visualize results and insights |

### 🌍 **Real-World Impact**
- **Urban Planners**: Data-driven city design decisions
- **Government Agencies**: Policy impact assessment
- **Researchers**: Urban development studies
- **Developers**: Sustainable project planning

### ⚡ **Installation Methods**

<details>
<summary><strong>🎯 Method 1: Standard Installation (Recommended)</strong></summary>

```bash
# 1. Clone the repository
git clone https://github.com/manish12ys/Neuro-Urban.git
cd Neuro-Urban

# 2. Create virtual environment (recommended)
python -m venv neurourban_env
source neurourban_env/bin/activate  # Linux/macOS
# OR
neurourban_env\Scripts\activate     # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Verify installation
python -c "from src.ml.city_analyzer import CityAnalyzer; print('✅ Installation successful!')"

# 5. Launch the application
streamlit run streamlit_app.py
```

</details>

<details>
<summary><strong>🪟 Method 2: Windows Easy Setup</strong></summary>

```batch

run_neurourban.bat

# This will automatically:
# - Check Python installation
# - Install dependencies
# - Launch the web interface
```

</details>

<details>
<summary><strong>⚡ Method 3: GPU-Accelerated Setup</strong></summary>

```bash
# 1. Verify CUDA installation
nvidia-smi

# 2. Install with GPU support
pip install -r requirements.txt

# 3. Verify GPU acceleration
python -c "import cupy as cp; print(f'✅ GPU acceleration available: {cp.cuda.is_available()}')"

python main.py --use-gpu
```

### 💻 **Python API Examples**

<details>
<summary><strong>📊 1. Collect and Analyze City Data</strong></summary>

```python
from src.config.settings import Config
from src.data.data_collector import CityDataCollector
from src.ml.city_analyzer import CityAnalyzer

# Initialize configuration
config = Config()

# Collect data from global cities
print("🌍 Collecting city data...")
collector = CityDataCollector(config)
city_data = collector.collect_all_data()
print(f"✅ Collected data for {len(city_data)} cities")

# Analyze cities with GPU acceleration
print("🧠 Analyzing cities...")
analyzer = CityAnalyzer(config)
results = analyzer.analyze_all_cities()

# Display results
print("\n🏆 Top 5 Cities by Category:")
for category, rankings in results['rankings'].items():
    print(f"\n{category.replace('_', ' ').title()}:")
    for i, (city, score) in enumerate(list(rankings.items())[:5], 1):
        print(f"  {i}. {city}: {score:.3f}")
```

</details>

<details>
<summary><strong>🏗️ 2. Generate City Blueprint</strong></summary>

```python
from src.dl.simple_blueprint_generator import SimpleBlueprintGenerator
import matplotlib.pyplot as plt
import base64
from io import BytesIO

# Initialize blueprint generator
generator = SimpleBlueprintGenerator(config)

# Define city parameters
city_params = {
    'population': 2_000_000,        # 2 million people
    'area_km2': 800,                # 800 square kilometers
    'sustainability_focus': True,    # Prioritize green spaces
    'tech_focus': True,             # Smart city features
    'cultural_focus': True,         # Preserve cultural sites
    'density': 'medium'             # Medium population density
}

# Generate blueprint
print("🏗️ Generating city blueprint...")
blueprint_data = generator.generate_blueprint(city_params)

# Display statistics
stats = blueprint_data['city_statistics']
print(f"\n📊 Generated City Statistics:")
print(f"Population Density: {stats['population_density']:.0f} people/km²")
print(f"Green Space per Capita: {stats['green_space_per_capita_m2']:.1f} m²/person")
print(f"Sustainability Score: {stats['sustainability_score']:.1f}/100")
print(f"Livability Score: {stats['livability_score']:.1f}/100")

# Save blueprint image
image_data = base64.b64decode(blueprint_data['blueprint_image'])
with open('generated_city_blueprint.png', 'wb') as f:
    f.write(image_data)
print("💾 Blueprint saved as 'generated_city_blueprint.png'")
```

</details>

## �️ Troubleshooting

<details>
<summary><strong>🐛 Common Issues & Solutions</strong></summary>

### **Import Errors**
```bash
# Issue: "ImportError: No module named 'cupy'"
# Solution: Install GPU dependencies
pip install cupy-cuda11x>=11.0.0

# Issue: "ImportError: No module named 'cuml'"
# Solution: Install via conda (optional)
conda install -c rapidsai cuml
```

### **GPU Issues**
```bash
# Issue: "CUDA out of memory"
# Solution: Reduce batch size or use CPU mode
export NEUROURBAN_NO_PYTORCH=1
python main.py

# Issue: "CUDA not detected"
# Solution: Verify CUDA installation
nvidia-smi
```

### **Windows-Specific Issues**
```batch
# Issue: Unicode/emoji display problems
# Solution: Use the provided batch file
run_neurourban.bat

# Issue: Path-related errors
# Solution: Run from project root directory
cd neurourban
python main.py
```

### **Performance Issues**
```bash
# Issue: Slow analysis
# Solution: Enable GPU acceleration
python -c "import cupy; print('GPU available:', cupy.cuda.is_available())"

# Issue: Memory errors
# Solution: Increase virtual memory or use smaller datasets
```
