### **Import Errors**
```bash
# Issue: "ImportError: No module named 'cupy'"
# Solution: Install GPU dependencies
pip install cupy-cuda11x>=11.0.0

# Issue: "ImportError: No module named 'cuml'"
```

### **GPU Issues**
```bash
# Issue: "CUDA out of memory"
# Solution: Reduce batch size or use CPU mode
export NEUROURBAN_NO_PYTORCH=1
python main.py

# Issue: "CUDA not detected"
# Solution: Verify CUDA installation
nvidia-smi
```

### **Windows-Specific Issues**
```batch
# Issue: Unicode/emoji display problems
# Solution: Use the provided batch file
run_neurourban.bat

# Issue: Path-related errors
# Solution: Run from project root directory
cd neurourban
python main.py
```
